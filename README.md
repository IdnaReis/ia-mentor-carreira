# 🌟 IA Mentor de Carreira: Descubra Seu Futuro em Tech

## 📌 Objetivo do Projeto
O **IA Mentor de Carreira** é um projeto que utiliza inteligência artificial para ajudar pessoas a planejarem e desenvolverem suas carreiras em tecnologia.  
A proposta é oferecer orientação personalizada, sugerir trilhas de aprendizado e indicar oportunidades de crescimento profissional.

---

## 🚀 Como Usar
1. Clone este repositório:
   ```bash
   git clone https://github.com/IdnaReis/ia-mentor-carreira.git
   
   ```
## 🛠️ Tecnologias Utilizadas

- Python → para lógica e processamento

- Git/GitHub → para versionamento e colaboração

- Visual Studio Code → como ambiente de desenvolvimento

- Inteligência Artificial → para recomendações de carreira
