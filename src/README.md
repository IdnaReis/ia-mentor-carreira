
# 📂 Código-Fonte da Aplicação

## 📌 Descrição
Esta pasta contém toda a lógica principal da aplicação IA Mentor de Carreira.

---

## 📁 Estrutura

- `main.py` → ponto de entrada  
- `career.py` → regras de negócio  
- `utils.py` → funções auxiliares  

---

## 🧠 Arquitetura
O projeto segue uma separação simples:

- Camada de entrada → `main.py`  
- Lógica de negócio → `career.py`  
- Suporte/utilitários → `utils.py`  

---

## 🎯 Objetivo
Manter o código organizado, testável e escalável.

---

## 🧪 Testabilidade
A estrutura foi pensada para facilitar testes unitários, principalmente nas funções de `career.py` e `utils.py`.
